# PeerLearn – Peer-to-Peer Learning Platform

PeerLearn is a collaborative web-based learning platform designed to connect students, self-learners, and educators in an interactive online environment. The platform allows users to ask questions, share learning resources, provide answers, and engage in academic discussions, promoting knowledge sharing and peer-to-peer learning.

Built using **Java (J2EE), JSP, Servlets, and MySQL**, PeerLearn follows the **MVC (Model-View-Controller)** architecture to ensure scalability, maintainability, and clean code organization.

---

## Features

### User Features

* User Registration & Login
* Profile Management
* Ask Questions & Share Learning Materials
* Create, Edit, and Delete Posts
* Like/Upvote Helpful Content
* Search & Filter Posts
* Participate in Discussions and Comments

### Admin Features

* Secure Admin Authentication
* User Management
* Content Moderation
* Platform Monitoring

### Security Features

* Role-Based Access Control
* Session Management
* Authentication & Authorization
* Form Validation
* URL Access Restriction Using Filters

---

## System Architecture

The application follows the MVC Architecture:

* **Model** – Handles business logic and database operations.
* **View** – User interface built using JSP.
* **Controller** – Servlets that process requests and responses.
